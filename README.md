#  Expired Domain Rebuilder & SEO Signal Analyzer

## Purpose
Analyze expired domains using **free SEO signals** to decide:
- Rebuild
- Redirect
- Avoid

## Signals used
- Archive.org snapshot depth
- DNS resolution
- Heuristic SEO scoring

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Cloudflare Pages + Workers compatible

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Directly deploy to Cloudflare Pages (Next.js preset).
